# 设置开发环境

由于 redi 需要使用 TypeScript 装饰器语法，因此你需要在 tsconfig.json 中启用 `experimentalDecorators`。特别注意：你需要保证是由 TypeScript 对你的源码进行翻译而不是 Babel，因为 Babel 对与装饰器的支持与 TypeScript 并不相同。

## 脚手架工具

听起来很麻烦？你可以直接使用我们提供的脚手架工具 [redi-starter](https://github.com/wendellhu95/redi-starter) ，redi-starter 除了启用了装饰器语法之外，还提供了下列功能：

* 支持按照平台进行代码拆分
* 集成 TypeScript, Prettier, ESLint, Webpack
