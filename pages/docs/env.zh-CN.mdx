# 设置开发环境

由于 redi 的 API 使用 TypeScript 装饰器语法，因此你需要在 tsconfig.json 中启用 `experimentalDecorators`。

```json highlight=3
{
  "compilerOptions": {
    "experimentalDecorators": true,
  }
}
```

<Callout>
特别注意：你需要保证是由 TypeScript 对你的源码进行翻译而不是 Babel，因为 Babel 对与装饰器的支持与 TypeScript 并不相同。
</Callout>

## 脚手架工具

你可以直接使用我们提供的脚手架工具 [redi-starter](https://github.com/wendellhu95/redi-starter) ，它提供了下列功能：

* 引入了 redi 并做好了相应的 TypeScript 配置
* 集成 Prettier / ESLint / webpack / Jest
* 支持按照不同的运行平台进行代码加载，例如手机上不会加载 `PcPlatformService` 的代码

### 使用

只需要将这个项目 clone 到本地：

```bash
git clone https://github.com/wendellhu95/redi-starter.git
```

如果你不想保留 Git 历史，可以使用 [degit](https://github.com/Rich-Harris/degit) 工具：

```bash
npx degit https://github.com/wendellhu95/redi-starter
```
